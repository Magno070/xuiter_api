# Xuiter API

A Node.js API built with NestJS.

## Requirements

- Node.js 20+
- npm 10+

## Getting Started

Install dependencies:

```bash
npm install
```

## Run

```bash
# development (watch)
npm run start:dev

# production
npm run build
npm run start:prod
```

## Test

```bash
# unit tests
npm test

# e2e tests
npm run test:e2e

# coverage
npm run test:cov
```

## Project Structure

- `src/` — application source code
- `test/` — e2e tests
- `dist/` — build output (after `npm run build`)
